Prodavnica-mini-cms
Mini-cms. Domaći u sklopu Django kursa. Sajt za prodaju skutera

Sadržaj:

početna stranica - /

stranica sa svim proizvodima - /scooters/

stranica sa detaljim informacijama za odredjini proizvod - /scooters/[id]/

Pokretanje projekta:
1. venv
2. install requirements
3. migrate
4. createsuperuser
5. runserver
